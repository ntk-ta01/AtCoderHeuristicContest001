## 現状の方針

* 外周に近い順から長方形の大きさを決める。増加可能できるスペースが多い方向を選んで増やす。他の長方形と交差しないか調べる。
* 焼きなまし法を書いた。8割の長方形について上下左右どちらかに拡大する。まあまあ解が動くようになった。
